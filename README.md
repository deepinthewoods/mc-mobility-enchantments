# Mobility Enchantments

A Fabric mod for Minecraft 1.21.10 that adds five unique mobility enchantments for chestplates.

## Features

- **Swooping**: Glide using your current velocity vector
- **Dash**: Burst forward in the direction you're looking
- **Double Jump**: Jump in mid-air
- **Elytra**: Elytra-like flight with reduced lift
- **Wall Jump**: Jump off walls with enhanced air control

All enchantments are mutually exclusive and require hunger to use.
